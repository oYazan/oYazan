## 🚀 Featured Hardware & Systems Projects

* **Custom Home Network Infrastructure & VPN Gateway:** Programmed a GL.iNet AR300M16 to act as a secure travel and home VPN gateway, utilizing custom DNS routing pipelines and dynamic hotspot routing to bypass ISP restrictions.
* **Manual EEPROM Programmer:** Designed a purely hardware-based circuit to manually flash and drive an EEPROM memory chip to control a 7-segment display, handling raw memory addressing and digital logic without microcontrollers.
* **Low-Latency Remote Workstation:** Engineered an optimized remote access server on a ThinkStation P520, utilizing hardware-level GPU encoding (NVENC) and bypassing CGNAT via encrypted UDP streaming.
